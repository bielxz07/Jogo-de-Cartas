 tema central do jogo: Um RPG estratégico de cartas onde você é um Guardião Elemental, escolhido para reunir os reinos divididos pelos elementos. Comande heróis, invoque criaturas e domine magias para restaurar o equilíbrio do mundo antes que o Caos Consumidor destrua tudo.

linguagem usada: html, css e JavaScript

Passo a Passo para Execução
Copie o código completo:

Selecione todo o código HTML fornecido (desde <!DOCTYPE html> até </html>)

Copie para a área de transferência (Ctrl+C ou Command+C)

Abra o VS Code, Se não tiver instale por esse site: https://code.visualstudio.com/

Inicie o Visual Studio Code no seu computador

crie uma pasta se não tiver, para ficar salvo

Crie um novo arquivo:

Clique em "File" > "New File" (ou Ctrl+N/Command+N)

Coloque o nome do arquivo com o final .html

Cole o código copiado (Ctrl+V ou Command+V)

Salve o arquivo:

Clique em "File" > "Save" (ou Ctrl+S/Command+S)

Nomeie o arquivo como confronto-elemental.html

Certifique-se de salvar com extensão .html

Execute o jogo:

Abra o arquivo salvo no seu navegador preferido:

Método 1: Clique com o botão direito no arquivo no VS Code e selecione "Open with Live Server" (se tiver a extensão Live Server instalada)

Método 2: Arraste o arquivo para uma janela do navegador aberta

Método 3: Clique com o botão direito no arquivo no explorador de arquivos e selecione "Abrir com" > seu navegador

Funcionalidades do Jogo
Menu Principal:

Iniciar Batalha: Começa um novo jogo

Construtor de Deck: Permite criar seu deck personalizado

Grimório de Regras: Explica como jogar

Crônicas do Jogo: Mostra informações sobre elementos e vantagens

Dificuldades:

Aprendiz (Fácil)

Mestre (Médio)

Arquimago (Difícil)

Como Jogar:

Selecione uma carta da sua mão clicando nela

Clique em "Finalizar Turno" para jogar a carta selecionada

O jogo compara automaticamente as cartas e determina o vencedor

O primeiro a alcançar 10 pontos vence

Personalização
Construtor de Deck:

Você pode criar um deck personalizado com até 18 cartas

As alterações são salvas automaticamente no navegador

Efeitos Visuais:

O jogo inclui animações de partículas e efeitos luminosos

Temas elementais com cores distintas para cada tipo de carta
